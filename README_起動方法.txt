===========================================
ChemEng - 化学工学計算ツール
===========================================

【起動方法】
1. ChemEng_Start.bat をダブルクリック
2. メニューが表示されます
3. [1] を選択して通常起動
4. 初回は自動セットアップが実行されます（数分かかります）
5. ブラウザが自動で開きます (http://localhost:8000)

【メインメニュー】
  [1] 起動（通常スタート）
      → 5つのステップで環境チェック＆サーバー起動
  [2] 修復（依存パッケージ再インストール）
      → パッケージが壊れた時に使用
  [3] リセット（仮想環境を削除して再構築）
      → 環境全体を作り直したい時に使用
  [4] ヘルプ・トラブルシューティング
  [5] 終了

【通常起動の5ステップ】
  Step 1/5: Python環境の確認
  Step 2/5: 仮想環境の準備
  Step 3/5: 依存パッケージの確認・インストール
  Step 4/5: ポートの確認
  Step 5/5: サーバー起動＆ブラウザ自動表示

  ※ 各ステップで問題があれば、原因と対処法が表示されます
  ※ 失敗したステップはリトライ可能です

【必要環境】
- Windows 10/11
- Python 3.10以上
  ※未インストールの場合: 起動時にインストール方法を案内します
  ※winget対応環境では自動インストールも可能です

【Linux/macOS での起動方法】
  bash start.sh               通常起動
  bash start.sh --port 8080   ポート指定
  bash start.sh --repair      依存パッケージ再インストール
  bash start.sh --reset       仮想環境を削除して再構築
  bash start.sh --help        ヘルプ表示

【機能一覧】
- VLE計算（気液平衡）
- 蒸留塔設計
- 熱交換器設計
- 配管圧力損失計算
- ポンプ動力計算
- 液液抽出
- ガス吸収
- LCOH計算（水素製造原価）
- ダッシュボード（結果比較）

===========================================
外部通信について
===========================================

【結論: 外部通信は不要】
本ツールはローカル完結型です。
インターネット接続なしで全機能が動作します。

【詳細】
- 計算処理: すべてローカルのPythonライブラリで実行
- 物性データ: thermo/chemicalsライブラリに30,000種以上内蔵
- Web UI: ローカルサーバー (localhost:8000) で動作
- データ保存: ブラウザのLocalStorageを使用

【外部通信が発生するケース（オプション）】
以下は本ツールの基本機能には含まれません:
- 初回セットアップ時のパッケージダウンロード（pip install）
- Vercelデプロイ時のプロキシ通信
- ngrokトンネル使用時
- AI機能拡張時（anthropicライブラリ、未実装）

===========================================
トラブルシューティング
===========================================

Q: 起動しない
A: メニューの [4] ヘルプを確認してください
   Pythonがインストールされているか確認:
   コマンドプロンプトで python --version を実行

Q: 計算できない・エラーが出る
A: メニューの [2] 修復 を試してください
   改善しない場合は [3] リセット を実行

Q: ポート8000が使用中
A: 起動時に自動で別ポート（8080等）が提案されます

Q: 特定の物質が見つからない
A: 英語名またはCAS番号で検索してください
   例: "ethanol", "64-17-5"

Q: ログファイルの場所
A: chemeng_setup.log（起動スクリプトと同じフォルダ）

【終了方法】
コマンドプロンプトウィンドウを閉じるか、Ctrl+C

===========================================
技術仕様
===========================================

【対応物質数】
- thermo/chemicals: 30,000種以上の有機・無機化合物
- CoolProp: 120種以上の冷媒
- Cantera: 燃焼・反応系化学種

【計算精度】
- VLE計算: NRTL, UNIFAC, SRK, PR 状態方程式対応
- 物性推算: グループ寄与法、対応状態原理
- 冷媒: REFPROP相当の高精度物性

【サポートOS】
- Windows 10/11 (推奨)
- macOS (動作確認済み)
- Linux (動作確認済み)
