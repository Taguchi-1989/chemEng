===========================================
ChemEng - 化学工学計算ツール
===========================================

【起動方法】
1. ChemEng_Start.bat をダブルクリック
2. 初回は自動セットアップが実行されます（数分かかります）
3. ブラウザが自動で開きます

【必要環境】
- Windows 10/11
- Python 3.9以上
  ※未インストールの場合: https://www.python.org/downloads/

【機能一覧】
- VLE計算（気液平衡）
- 蒸留塔設計
- 熱交換器設計
- 配管圧力損失計算
- ポンプ動力計算
- 液液抽出
- ガス吸収
- LCOH計算（水素製造原価）
- ダッシュボード（結果比較）

【トラブルシューティング】
Q: 起動しない
A: Pythonがインストールされているか確認してください

Q: 計算できない
A: venvフォルダを削除して再起動（依存関係を再インストール）

【終了方法】
コマンドプロンプトウィンドウを閉じるか、Ctrl+C
