===========================================
ChemEng - 化学工学計算ツール
===========================================

【起動方法】
1. ChemEng_Start.bat をダブルクリック
2. 初回は自動セットアップが実行されます（数分かかります）
3. ブラウザが自動で開きます (http://localhost:8000)

【必要環境】
- Windows 10/11
- Python 3.9以上
  ※未インストールの場合: https://www.python.org/downloads/

【機能一覧】
- VLE計算（気液平衡）
- 蒸留塔設計
- 熱交換器設計
- 配管圧力損失計算
- ポンプ動力計算
- 液液抽出
- ガス吸収
- LCOH計算（水素製造原価）
- ダッシュボード（結果比較）

===========================================
外部通信について
===========================================

【結論: 外部通信は不要】
本ツールはローカル完結型です。
インターネット接続なしで全機能が動作します。

【詳細】
- 計算処理: すべてローカルのPythonライブラリで実行
- 物性データ: thermo/chemicalsライブラリに30,000種以上内蔵
- Web UI: ローカルサーバー (localhost:8000) で動作
- データ保存: ブラウザのLocalStorageを使用

【外部通信が発生するケース（オプション）】
以下は本ツールの基本機能には含まれません:
- Vercelデプロイ時のプロキシ通信
- ngrokトンネル使用時
- AI機能拡張時（anthropicライブラリ、未実装）

===========================================
トラブルシューティング
===========================================

Q: 起動しない
A: Pythonがインストールされているか確認してください
   コマンドプロンプトで python --version を実行

Q: 計算できない
A: venvフォルダを削除して再起動（依存関係を再インストール）

Q: ポート8000が使用中
A: 他のアプリを終了するか、server.py --port 8080 で別ポート

Q: 特定の物質が見つからない
A: 英語名またはCAS番号で検索してください
   例: "ethanol", "64-17-5"

【終了方法】
コマンドプロンプトウィンドウを閉じるか、Ctrl+C

===========================================
技術仕様
===========================================

【対応物質数】
- thermo/chemicals: 30,000種以上の有機・無機化合物
- CoolProp: 120種以上の冷媒
- Cantera: 燃焼・反応系化学種

【計算精度】
- VLE計算: NRTL, UNIFAC, SRK, PR 状態方程式対応
- 物性推算: グループ寄与法、対応状態原理
- 冷媒: REFPROP相当の高精度物性

【サポートOS】
- Windows 10/11 (推奨)
- macOS (動作確認済み)
- Linux (動作確認済み)
